# Wait-Debugger

## SYNOPSIS
Stops a script in the debugger before running the next statement in the script.

## SYNTAX

```
Wait-Debugger [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```
