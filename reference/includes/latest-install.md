> [!NOTE]
> The installation commands in this article are for the latest stable release of PowerShell. To
> install a different version of PowerShell, adjust the command to match the version you need. Open
> the [Release tags page][01] on GitHub. Select the version you want to install. The download links
> for every package are found in the **Assets** section of the release. The **Assets** section may
> be collapsed, so you may need to click to expand it.
